# 幂和公式的多项式规律

## 基本信息

- 标题：幂和公式，能不能自己算出来？
- 主题 slug：`power-sum-polynomials`
- 目标观众：知道等差数列求和、看得懂简单多项式展开的高中及以上观众。
- 视频长度：低清预览目标约 2 分 40 秒到 3 分钟。
- 核心问题：自然数幂的前 n 项和为什么总像一个多项式？能不能不用背公式，而是用待定系数法把它算出来？
- 一句话结论：如果一个求和多项式每往前走一步的增量正好是 `n^p`，那它就能通过“差分等于当前项”来锁定系数；三次方求和会自然变成三角数的平方。

## 观众承诺

看完后，观众会明白幂和公式不是一堆孤立记忆点，而是一种“反过来找多项式”的方法：先猜次数，再让相邻两项之差等于要加上的那一项。

## 理解风险与桥接

- 风险：观众不知道为什么可以先猜多项式。
  - 桥接：强调“我们不是乱猜，而是因为差分会把多项式次数降一阶”。
- 风险：`P(n)-P(n-1)` 展开量大，容易跟丢。
  - 桥接：只展示四个基础差分，再用颜色把 `a,b,c,d` 的贡献合并。
- 风险：伯努利数显得突然。
  - 桥接：把它讲成“当这个系数账本一直做下去，出现的一串固定数字”。
