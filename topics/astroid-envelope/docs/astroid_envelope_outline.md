# 星形线包络视频大纲

## 主题

从“靠墙滑落的棍子扫过区域边界”这个实际场景出发，用包络线和微分推导第一象限星形线：

```text
x^(2/3) + y^(2/3) = L^(2/3)
```

## 观众钩子

开头不直接讲星形线，而是问：

“一根靠墙滑落的棍子扫出的边界，会是圆弧吗？”

这能制造反直觉：真实答案不是圆弧，而是一条星形线。

## 比喻

- 棍子的每一瞬间：一条直线。
- 很多直线叠在一起：像一把扫帚在墙角扫出痕迹。
- 边界曲线：不是某一根棍子画出来的，而是所有棍子共同“擦”出来的包络线。
- 微分条件：相邻两根棍子在边界处几乎重合，所以对角度的一阶变化为零。

## 推导节奏

1. 展示棍子滑落。
2. 显示多根半透明位置线。
3. 高亮这些线的包络边界。
4. 参数化端点：`A=(L cosθ,0)`，`B=(0,L sinθ)`。
5. 写出直线族：`F=x secθ+y cscθ-L=0`。
6. 加入包络条件：`∂F/∂θ=0`。
7. 联立得到参数式：`x=L cos³θ, y=L sin³θ`。
8. 消去 θ 得到星形线方程。
